Change Password

Open command line. Go to the Jetty folder (Windows default: C:\Program Files\repox\jetty6.1-11 ; Linux is [UNCOMPRESSED_REPOX_FOLDER]/jetty6.1-11 )

Execute:
[WINDOWS]> bin\generatePassword.bat . [USER] [PASSWORD]
[LINUX]> bin/generatePassword.bat . [USER] [PASSWORD]
