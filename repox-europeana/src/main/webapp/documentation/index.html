REPOX is a standalone OAI-PMH server developed in WP2 of the TELplus project. It aims to provide all The European Library partners with a simplified and flexible solution for exposing their data via OAI-PMH. The data below is very important and you should keep it at hand in case you need assistance (see Contacts).

This page is generated on the installation process of REPOX. If this installation had problems or was manual, the configurations presented (local page and directories) are incorrect.

The page of your local REPOX installation is: http://localhost:$repox.jettyPort/repox

The Jetty Server directory is $repox.jetty/jetty-6.1.11/
The REPOX Repository (where the records are saved) is $repositorydir
The REPOX configuration dir is in $configurationdir
The REPOX database is in $databasedir
